# Writing Skills

日本語の技術文書・書籍原稿を書く、または推敲するための skill 群です。
文章規範、段落構成、論証の厳密さ、読み手の負荷管理を扱います。

## どの Skill を使うか

- 日本語の技術書・記事・解説文を書く → [`japanese-tech-writing`](./japanese-tech-writing/SKILL.md)
- 草稿を一文一行や段落構成の規範に沿って推敲する → [`japanese-tech-writing`](./japanese-tech-writing/SKILL.md)
- LLM っぽい空句や冗長な言い回しを削る → [`japanese-tech-writing`](./japanese-tech-writing/SKILL.md)

## 典型フロー

1. 執筆・推敲対象の読者、目的、媒体を確認する。
2. `japanese-tech-writing` の規範に沿って構成・論証・文体を点検する。
3. 必要に応じて一文一行、引用、脚注、コラム記法などの整形規則を適用する。

## Skill 一覧
