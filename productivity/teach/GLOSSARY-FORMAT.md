# GLOSSARY.md Format

`GLOSSARY.md` は teaching workspace の標準用語集である。説明、練習、learning record はこの用語に合わせる。概念を短い定義へ圧縮できること自体が理解の証拠になる。

## Structure

```md
# {Topic} Glossary

{この glossary が扱う topic の説明を 1-2 文で書く。}

## Terms

**Hypertrophy**:
Mechanical tension と metabolic stress が繰り返されることで起こる muscle growth。
_Avoid_: Bulking, getting big

**Progressive overload**:
Load、volume、intensity などを通じて、muscle への要求を時間とともに体系的に増やすこと。
_Avoid_: Pushing harder, levelling up

**RPE (Rate of Perceived Exertion)**:
Set の主観的きつさを 1-10 で表す rating。10 は failure、8 はあと 2 reps 余る状態。
_Avoid_: Effort score, intensity rating
```

## Rules

- **ユーザーが理解した用語だけ追加する。** glossary は辞書ではなく圧縮された学習記録である。初出概念は、ユーザーが正しく使えるまで昇格しない。
- **opinionated にする。** 同じ概念に複数語がある場合は最適な語を選び、避ける alias を列挙する。
- **定義は短くする。** 1-2 文。何であるかを定義し、やり方の説明にしない。
- **glossary 内の用語を使って定義する。** 一度採用した用語は他の定義でも優先する。
- **自然な cluster ができたら見出しで分ける。** 最初は flat list でよい。
- **曖昧さを明示する。** 広い分野で雑に使われる語は、この workspace での意味を決める。
- **理解が深まったら更新する。** 古い定義を放置しない。
